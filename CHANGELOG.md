# Changelog

## 0.1.0 - 2026-05-10

- Introduced the project as PiSight.
- Added a Raspberry Pi focused face recognition package.
- Added CLI commands for setup checks, sample collection, model training, and live recognition.
- Added the `pisight` command, with `raspberry-face-recognition` kept as a compatibility alias.
- Added JSON configuration, local dataset helpers, and OpenCV integration helpers.
- Added Raspberry Pi setup notes, usage documentation, a systemd service example, and unit tests.
- Added GitHub Actions workflow for Python 3.9 through 3.12.
